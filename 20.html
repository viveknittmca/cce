1 An inference engine based on Forward Chaining searches the
Inference rules until it finds one where the IF clause is
Known to be TRUE.
When found it can conclude. Else it can Infer the THEN clause
resulting in addition of new Information to the data set.
Hence, it is called as Data Driven Approach.		 An inference engine based on Backward Chaining
searches the inference rules until it finds a THEN clause
Matching the desired GOAL.
If the IF clause is NOT TRUE then it is added to the list of goals.
Hence, it is called as Goal Driven Approach.
